tips & tricks

check out the resources below and learn the sweet 16 TOPs, CHOPs, SOPs and DATs

  • operators (tab)
    • TOPs = texture operators i.e. 2d images / movies etc
    • CHOPs = channel operators i.e.
    • SOPs = surface operators i.e. 3d surfaces or geometry
    • COMPs = components which can contain networks
    • MATs = material operators i.e. shaders for 3d geometry
    • DATs = data operators i.e. tables, json, web APIs
